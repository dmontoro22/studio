# **App Name**: TFG Entrenamiento

## Core Features:

- Autenticación de Usuario: Implementar un sistema de registro y login de usuarios seguro utilizando Firebase Authentication para gestionar el acceso a la aplicación.
- Formulario de Registro de Ejercicio: Un formulario intuitivo para que los usuarios puedan registrar sus ejercicios, incluyendo campos para Ejercicio, Peso, Repeticiones y Esfuerzo Percibido (RPE).
- Almacenamiento de Datos en Firestore: Almacenar de forma segura los registros de entrenamientos (ejercicio, peso, repeticiones, RPE, fecha) en Cloud Firestore.
- Historial de Entrenamiento: Una página dedicada a mostrar un historial cronológico de todos los entrenamientos registrados, obteniendo los datos desde Cloud Firestore.
- Herramienta de Sugerencia de Ejercicios: Una herramienta generativa impulsada por IA que sugiere variaciones de ejercicios o nuevas rutinas basadas en el historial de entrenamiento del usuario y sus objetivos.

## Style Guidelines:

- Esquema de color oscuro (Dark Mode). El color primario será un azul profesional (#7094DB), que aporta una sensación de confianza y modernidad en entornos oscuros.
- El color de fondo será un gris oscuro sutilmente azulado (#1A1D23), que establece una base profunda y profesional para la interfaz de usuario.
- Un acento vibrante en cian (#8CE2F2) para resaltar elementos interactivos y la información clave, proporcionando un contraste enérgico y claro.
- Toda la tipografía utilizará la fuente 'Inter', un tipo de letra sans-serif moderno y de alta legibilidad, adecuado para interfaces de usuario limpias y profesionales.
- Uso de iconos minimalistas y lineales que transmitan claramente la función y mejoren la estética moderna de la aplicación.
- Un diseño espacial y limpio con énfasis en la jerarquía visual clara. Los datos se presentarán en tarjetas o módulos bien definidos, adaptándose a diferentes tamaños de pantalla para una experiencia responsiva.
- Animaciones sutiles y fluidas para transiciones de página, estados de botones y confirmaciones de acciones, proporcionando una experiencia de usuario refinada sin distracciones.
